Свойство BackGround - задний фон

Фон элемента HTML — это то, что появляется за текстом. Хотя CSS позволяет применять фон к элементу любого типа, он в основном используется для блочных элементов

Пример 1

Использование свойства background: "цвет";

Код CSS

.example-1 { background: #32b5a4; }

Пример 2

Использование background: rgb(a, b, c); и background: rgba(a, b, c, d);

Код CSS

.example-2 { background: rgb(0, 0, 0); } .example-3 { background: rgba(0, 0, 0, 0.5); }

Пример 3

Использование background-image: url('Путь к картинке'); или же background: url('Путь к картинке');,

Код CSS

.example-4 { background-image: url("img/bg.jpg"); }

Пример 4

Использование background-image: url('Путь к картинке'); + background-repeat: no-repeat;

Код CSS

.example-5{ background-image: url("img/avatar.jpg"); background-repeat: no-repeat; }

Пример 5

Использование background-image: url('Путь к картинке'); + background-position: center;

Код CSS

.example-6{ background-image: url("img/avatar.jpg"); background-repeat: no-repeat; background-position: center; }

Пример 6

Использование background-image: url('Путь к картинке'); + background-size: 'свойство';

Код CSS

.example-7{ background-image: url("img/avatar.jpg"); background-repeat: no-repeat; background-size: 'contain' или 'cover'; }

Пример 7

Использование background-image: url('Путь к картинке'); + background-attachment: 'свойство';

Код CSS

.example-8{ background-image: url("img/avatar.jpg"); background-repeat: no-repeat; background-attachment: 'fixed' или 'local'; }

Доподнительные штуки (градиенты, универсальное свойство background)

Пример 8

Использование универсального свойства background: url('Путь к картинке') 0 0 / cover no-repeat;

Код CSS

.example-9{ background: url('img/bg.jpg') 0 0 / cover no-repeat; }

Пример 9

Использование линейного градиента

Код CSS

.example-10{ background: linear-gradient(90deg, rgba(2,0,36,1) 0%, rgba(9,9,121,1) 35%, rgba(0,212,255,1) 100%); }

Пример 10

Использование радиального градиента

Код CSS

.example-11{ background: linear-gradient(90deg, rgba(2,0,36,1) 0%, rgba(9,9,121,1) 35%, rgba(0,212,255,1) 100%); }
